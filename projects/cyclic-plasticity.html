Machine Learning Modelling of Cyclic Plasticity

A neural-network surrogate for history-dependent cyclic material behaviour.

← Back to portfolio

Overview

This project explores the use of machine learning to reproduce cyclic plasticity behaviour from sequential material-state information.

Model formulation

The network uses a lookback window of previous loading states to predict the increment of plastic strain. Stress is then reconstructed through the constitutive relation, enabling comparison between predicted and reference hysteresis loops.

Optimisation

Optuna is used to search over hyperparameters including learning rate, network depth, hidden units, and activation functions. Model performance is assessed using mean squared error and qualitative agreement of the cyclic response.

Research relevance

The project provides a foundation for data-driven constitutive modelling, scientific machine learning, and future hybrid physics–ML methods.
